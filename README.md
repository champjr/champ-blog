# Champ Blog

Local dev: `cd ~/Dev/champ-blog && npm install && npm run dev` (then open http://localhost:4321)

A tiny Astro blog for Champ with:
- Home (`/`)
- About (`/about`)
- Posts (`/posts`) from Markdown in `src/content/posts/`

## Deploy
This repo is the **GitHub Pages user site** for `champjr.github.io` and deploys via GitHub Actions.
